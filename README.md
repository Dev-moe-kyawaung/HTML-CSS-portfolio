## 🚀 Installation

Getting started with this portfolio is straightforward. No complex build tools or dependencies are required.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Dev-moe-kyawaung/HTML-CSS-portfolio.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd HTML-CSS-portfolio
    ```
3.  **Open in your browser:**
    Simply open the `index.html` file in your preferred web browser.
    ```bash
    open index.html # On macOS
    start index.html # On Windows
    xdg-open index.html # On Linux
    ```
    Alternatively, you can drag and drop the `index.html` file into your browser.
